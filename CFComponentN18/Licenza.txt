http://www.microsoft.com/it-IT/download/details.aspx?id=30709


Utilizzo del componente:
File del componente: CFComponentN18.dll
File di dati SQLServe CE 4 : dataX_2018.sdf
Decomprimere il pacchetto zip in una cartella. Installare i componenti (nel file CFComponentN18.dll) nella ToolBox in Visual Studio tramite la funzione "Scegli elementi".

Distribuzione:

Distribuire i file:
CFComponentN18.dll
System.Data.SqlServerCe.dll
dataX_2018.sdf

** Attenzione: **
il file dataX_2018.sdf deve essere posizionato di default nella directory:

C:\Users\user_name\AppData\Local

** ** ** ** ** **
cambiare questa impostazione qualora necessario.
